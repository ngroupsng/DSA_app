Linear Search — Visualizer

Overview

Linear Search checks each element in the array sequentially until the target element is found or the array ends.

  • Best Case: O(1) — first element found
  • Worst Case: O(n) — last or not found
  • Use Case: Works on unsorted arrays

Pseudocode

LinearSearch(arr, n, key):
  for i = 0 to n-1:
    if arr[i] == key:
      return i
  return -1
