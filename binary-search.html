Overview

Binary Search works on sorted arrays. It repeatedly divides the search interval in half until the target is found or the interval is empty.

  • Best Case: O(1)
  • Average Case: O(log n)
  • Worst Case: O(log n)

Pseudocode

BinarySearch(arr, key):
  low = 0
  high = n - 1
  while low <= high:
    mid = (low + high) / 2
    if arr[mid] == key:
      return mid
    else if arr[mid] < key:
      low = mid + 1
    else:
      high = mid - 1
  return -1
