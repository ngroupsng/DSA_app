Selection Sort Visualizer

Overview

Selection Sort is a simple sorting algorithm that repeatedly selects the minimum element from the unsorted part and moves it to the beginning. It divides the array into sorted and unsorted regions.

Complexity: O(n²) for all cases. In-place but not stable.

Pseudocode

for i = 0 to n-1
  min_index = i
  for j = i+1 to n-1
    if arr[j] < arr[min_index]
      min_index = j
  swap(arr[i], arr[min_index])
