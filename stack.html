Overview

A Stack is a linear data structure that follows the LIFO (Last In, First Out) principle. Elements are added and removed from the top only.

  • Push: Add an element to the top.
  • Pop: Remove the top element.
  • Peek: View the top element without removing it.

Pseudocode

PUSH(stack, x):
  if stack is full:
    print "Overflow"
  else:
    top = top + 1
    stack[top] = x

POP(stack):
  if stack is empty:
    print "Underflow"
  else:
    x = stack[top]
    top = top - 1
    return x
